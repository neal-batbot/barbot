# Platform Docs

This is the unified documentation entry for the platform phase-one integration.

## Integrated Products

- `pi-mono`: Agent experience and capability surface
- `supabase-ssh`: Web entry and SSH temporary credential flow
- `fumadocs`: Documentation site and gated premium content

## Unified Control Plane

`barbot` is the single source for:

- session identity
- bridge token issuance by audience
- entitlement lookup for product plans and features

## Entitlement Model

Entitlements are normalized to the same shape:

- `product`
- `plan`
- `features`
- `quota`
- `expiresAt`
